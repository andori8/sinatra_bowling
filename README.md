# Bowling Inventory

Welcome to the Sinatra Bowling Inventory. Here you can add balls to your personal inventory after logging in
with details about its weight, brand, etc.

## Usage

To use this app, just clone, run rake db:migrate and then run shotgun. Everything should be set up.
